# Dog Sitter Booking App

Welcome to the Dog Sitter Booking App! This iOS application is designed to streamline the process of booking dog sitters for your beloved pets. Whether you're a dog owner in need of a reliable sitter or a dog sitter looking to manage your bookings effectively, this app has you covered.

## Features

- **Booking Management**: Easily schedule and manage bookings with a user-friendly calendar interface.
- **Dog Breed Information**: Access detailed information about different dog breeds to help determine compatibility with your pet.
- **Profile Management**: Create and update your profile with information about your services or your pet's needs.

## Inspiration

This app was inspired by the personal experience of the developer as a dog sitter. Recognizing the need for a more efficient and convenient way to manage dog sititng gigs and learn more about the dogs cared for.

## Technologies Used

- Swift
- UIKit
- CoreData (for local data persistence)

## How to Use

1. Clone the repository to your local machine.
2. Open the project in Xcode.
3. Build and run the project on your iOS device or simulator.
